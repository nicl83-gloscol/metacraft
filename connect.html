Connection Details

MetaCraft

MetaCraft is currently running on Minecraft version 1.14.4.

You can join MetaCraft by connecting to the server at [FICTIONAL WEBSITE NOT REAL], either by adding it to your server list, or by using direct connection.

When you join MetaCraft for the first time, you will be greeted with a tutorial that explains how to use the protection plugins available on the server. You can always repeat the tutorial by running /tutorial.

MetaCraft Classic

MetaCraft Classic runs on Minecraft version Beta 1.7.3 - we do not intend to upgrade beyond this version for MetaCraft Classic.

When you make a donation to our server, you will need your Minecraft username to hand. This is so we can add your username to the whitelist for MetaCraft Classic.

The connection details for MetaCraft Classic will be sent to the email address associated with your PayPal account - if you have lost this email please contact the administrator.
