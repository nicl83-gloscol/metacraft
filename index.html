About Us

MetaCraft is a survival multiplayer server for Minecraft designed to be as close to vanilla as possible whilst still maintaining control and rule over the server.

We run Bukkit with very few plugins - largely plugins for managing world protection, anti-grief and anti-cheat. There is no economy, no purchasable kits, and no factions - just pure survival as Mojang intended.

Connection details can be found on the "Connection Details" tab at the top of the page.

We do accept donations - however, do understand that donations do NOT give you any in-game benefit. They merely make your name shiny when you talk in chat!

In addition, donating gives you access to MetaCraft Classic - a seperate server running Minecraft Beta 1.7.3. There is no minimum donation required - however, as MetaCraft Classic is designed to be a niche server, donations are required in order to keep it from becoming merely a secondary server for when MetaCraft is full.
